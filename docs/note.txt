# TTS API 接口服务说明

## /health接口

## /v1/audio/speech接口描述
在现有命令行添加参数 --openai,实现一个个openai标准协议的speech TTS接口:
/v1/audio/speech

### 生成TTS语音Curl示例
curl -X POST http://localhost:3003/v1/audio/speech \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "input": "The sun sets behind the mountains, painting the sky in shades of pink and orange.|The weather is beautiful and sunny outside. A gentle breeze makes the air feel fresh and pleasant.",
  "voice": "F1",  
  "model": "supertts"
  "speed": 1.0,  
}' --output output.wav

#### 请求参数说明
 - input: 输入的文字内容（目前仅支持英文）
 - voice 的值 F1、F2、M1、M2
 - model 的值 supertts
 - speech 推荐值 0.9-1.5

## 服务配置文件添加 config.json
{
    "onnx-dir": "assets/onnx",
    "total-step": 5,
    "voice-dir": "assets/voice_styles",
    "speed": 1.0
